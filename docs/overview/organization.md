## Publishing Services Overview

The Permissions Seeking function operates within Elsevier's Health Content Management (HCM) division:

- **Total Organization FTE**: 60+
- **Permissions Seeking FTE**: 19
- **Processes Handled**: 12 different processes

## Divisional Structure

```
Health Content Management (HCM)
├── Health Sciences (HS) Permissions
│   └── Copyrights Specialists
│
└── Science & Technology (S&T) Permissions
    └── Copyrights Specialists
```

## Workflows Supported

### Health Sciences (HS)
- HS (Health Sciences)
- Clinics
- RREF
- Translations
- APAC
- Nursing
- Journals
- E-products (IPOC, Mosby Skills, Clinical Key Essentials, etc.)

### S&T Workflows
- RREF
- Hot Topics
- MRW (Major Reference Works)

## Cross-Functional Collaboration

### Internal Partners

| Department | Collaboration Type |
|------------|-------------------|
| **Editorial** | Content requirements, timeline coordination |
| **Production** | Asset delivery, file formats |
| **Legal** | Complex rights issues, disputes |
| **Finance** | Payment processing, budget management |
| **Technology** | System support, integrations |

### External Partners

| Partner Type | Relationship |
|--------------|--------------|
| **CCC (Copyright Clearance Center)** | License clearing, RightsLink |
| **Publishers** | Rights negotiations, bulk agreements |
| **Authors** | Direct permission requests |
| **Societies** | Member content licensing |

## Communication Channels

### Internal

- **Email** - Primary communication
- **Teams/Slack** - Quick questions, team collaboration
- **RPC Comments** - Project-specific discussions
- **Weekly Meetings** - Team syncs, escalation reviews

### External

- **Email** - Formal permission requests
- **Publisher Portals** - Online license systems
- **Phone** - Escalations, urgent requests
- **CCC RightsLink** - Automated licensing
