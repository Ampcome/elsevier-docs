# CCC RightsLink

## Purchase Process Overview

![CCC RightsLink Purchase Flow](/diagrams/18_CCC_RightsLink_Purchase_Flow.svg)

## Overview

**Copyright Clearance Center (CCC) RightsLink** is an online licensing platform that enables quick purchase of permissions from participating publishers.

## Key Benefits

| Benefit | Description |
|---------|-------------|
| **Speed** | Instant license generation |
| **Convenience** | Online 24/7 availability |
| **Clarity** | Clear pricing upfront |
| **Documentation** | Automatic license records |

## When to Use RightsLink

### Ideal Scenarios

- Publisher participates in RightsLink
- Standard usage terms acceptable
- Timeline is urgent
- Price is within budget

### Not Recommended When

- Custom terms needed
- Bulk discount desired
- Price seems excessive
- Publisher prefers direct contact

## Using RightsLink

### Step-by-Step Process

1. **Navigate** to www.copyright.com
2. **Search** by title, DOI, or ISBN
3. **Select** the specific content
4. **Choose** reuse type and scope
5. **Review** terms and price
6. **Purchase** using company account
7. **Download** license confirmation

### Search Options

| Method | Best For |
|--------|----------|
| **Title** | Books, general search |
| **DOI** | Journal articles |
| **ISBN** | Book chapters |
| **ISSN** | Journal issues |
| **Author** | When DOI unknown |

### Reuse Types

| Type | Description |
|------|-------------|
| **Republish in a book** | Include in new publication |
| **Republish in journal** | Journal article reuse |
| **Academic/educational** | Teaching materials |
| **Digital** | Online publication |
| **Print** | Physical publication |
| **Print + Digital** | Combined rights |

## Pricing Factors

### What Affects Cost

| Factor | Impact |
|--------|--------|
| **Publisher** | Each sets own prices |
| **Content Type** | Figures vs. text varies |
| **Usage Scope** | World vs. regional |
| **Duration** | Time-limited vs. perpetual |
| **Format** | Print, digital, or both |
| **Commercial** | Higher than educational |

### Pricing Notes

- Prices vary significantly by publisher
- Price displayed as "PRICE: Pending" until selection is complete
- Currency selection: USD - $
- Use SAVE QUOTE button to save pricing for review

## Best Practices

### Before Purchasing

1. **Verify** exact content needed
2. **Confirm** usage scope matches needs
3. **Compare** to direct publisher pricing
4. **Check** if bulk discount available
5. **Get** approval if needed

### Documentation

## Troubleshooting

### Common Issues

| Issue | Solution |
|-------|----------|
| Content not found | Try different search terms, contact publisher |
| Price seems wrong | Clear cart, search again |
| License not received | Check spam, contact CCC |
| Wrong terms selected | Contact CCC support |

## Integration with Workflow

### From Permission Request to RightsLink

1. Receive permission request in RPC
2. Research indicates RightsLink available
3. Complete RightsLink purchase
4. Document in RPC with license details
5. Close item as Granted

### When RightsLink Doesn't Work

If RightsLink fails:
- Content not available → Contact publisher
- Price too high → Negotiate directly
- Terms inadequate → Request custom license
- Technical error → Try again or contact CCC
