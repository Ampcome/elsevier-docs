## ORCID

### What is ORCID?

- Unique 16-digit identifier for researchers
- Persistent digital identifier
- Links authors to their work
- Free to use

### Using ORCID for Permissions

1. Search orcid.org by author name
2. Find ORCID iD
3. View linked publications
4. Access contact information (if public)

### Benefits

| Benefit | How It Helps |
|---------|--------------|
| **Disambiguation** | Distinguish same-name authors |
| **Currency** | Often updated by authors |
| **Linking** | Connect to publications |
| **Contact** | May include email |

## PubMed

### Medical Content Search

| Feature | Purpose |
|---------|---------|
| **PMID** | Unique article identifier |
| **MeSH Terms** | Subject classification |
| **Author Search** | Find publications |
| **Affiliation** | Institution details |

### Using PubMed

1. Search by title, author, or PMID
2. Find article details
3. Note DOI for RightsLink
4. Check PMC for open access

### PubMed Central (PMC)

- Free full-text archive
- May indicate OA status
- Check license terms
- Could qualify for Tier 0

|-------------|---------|

### Policy Colors

| Color | Meaning |
|-------|---------|
| **Green** | Can archive pre/post-print |
| **Blue** | Can archive post-print |
| **Yellow** | Can archive pre-print |
| **White** | No archiving allowed |

## Open Access Resources

### Checking OA Status

| Resource | Purpose |
|----------|---------|
| **Unpaywall** | Browser extension for OA |
| **DOAJ** | OA journal directory |
| **OpenDOAR** | OA repository directory |
| **CORE** | OA research aggregator |

### License Verification

When content appears OA:
1. Verify license type (CC-BY, CC-BY-NC, etc.)
2. Check version (preprint vs. published)
3. Confirm matches your use case
4. Document for Tier 0 classification

## Best Practices

### Research Workflow

1. **Start** with DOI/PMID if available
2. **Cross-reference** multiple sources
3. **Verify** current affiliation
4. **Document** search attempts
5. **Save** source links

### Documentation

For each search:
- Record databases checked
- Note date of search
- Save relevant screenshots
- Document dead ends too
