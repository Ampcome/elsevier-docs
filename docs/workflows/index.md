## High-Level Process Stages

### Stage 1: Request Initiation

1. **Project Creation** - New book/journal project entered in RPC
2. **Rights Items Identified** - Third-party content flagged for permissions
3. **Initial Assessment** - Tier classification assigned

### Stage 2: Rights Holder Identification

1. **Source Analysis** - Examine the original publication
2. **Ownership Research** - Determine current rights holder
3. **Contact Discovery** - Find appropriate contact method

### Stage 3: Permission Request

1. **Channel Selection** - Choose contact method (email, portal, CCC)
2. **Request Submission** - Send permission request
3. **Documentation** - Log request in RPC

### Stage 4: Follow-up & Negotiation

1. **Response Tracking** - Monitor for replies
2. **Follow-up Cadence** - Systematic reminder schedule
3. **Negotiation** - Address terms, fees, conditions

### Stage 5: Resolution & Closure

1. **Permission Granted** - Document terms in RPC
2. **Permission Denied** - Notify editorial, find alternatives
3. **Closure** - Update project status, archive documentation

## Key Decision Points

| Decision | Options | Determines |
|----------|---------|------------|
| **Tier Classification** | Tier 0-4 (HS) or Tier 1-3 (S&T) | Workflow path |
| **Mode of Contact** | Author, Publisher, CCC, Portal | Contact strategy |
| **Fee Approval** | Pay, Negotiate, Escalate, Decline | Cost management |
| **Denial Handling** | Alternative source, Remove, Re-request | Content resolution |

## Workflow Variations

### By Content Division

| Division | Primary Workflow | Characteristics |
|----------|-----------------|-----------------|
| **Health Sciences** | Tier 0/4/Rule of 10 | High volume, standard medical content |
| **S&T** | Tier 1/2/3 | Lower volume, complex scientific content |

### By Content Type

Different content types require different handling approaches:

![Content Type Routing](/diagrams/21_Content_Type_Routing.svg)

| Content Type | Typical Path | Considerations |
|--------------|--------------|----------------|
| **Figures/Charts** | Standard flow | Most common type |
| **Tables** | Standard flow | Data accuracy critical |
| **Photos/Images** | May need model releases | Privacy concerns |
| **Text Excerpts** | Word count matters | STM guidelines apply |
| **Multimedia** | Special handling | Format considerations |
